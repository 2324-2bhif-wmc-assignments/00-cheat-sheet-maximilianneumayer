Lists

Unstyled List: <ul class="list-unstyled">
Nested Lists (Listen die weiter drinnen sind) werden nicht beeinflusst.
Inline List: <ul class="list-inline">
Item: <li class="list-inline-item">This is a list item.</li> (muss man machen, dass inline list funktioniert)
List:
<ul class="list-group">
<div class="list-group">
Items:
Normal Items: <li class="list-group-item">
Active Items: <li class="list-group-item active">
Disabled Items: <li class="list-group-item disabled">

Button in div list: <button type="button" class="list-group-item list-group-item-action (word)">
word: active or disabled or just write nothing, leave it blank
a in div list: <a href="#" class="list-group-item list-group-item-action">

Works with <button> and <a> as well. They also have the class .list-group-item-action where you get a hover effect.

Farbe:
<a class="list-group-item list-group-item-{color}" href="#">
color: primary, secondary, success, danger, warning, info, light, dark

Carousel

Carousel: <div class="carousel slide" id="carouselExampleCaptions">
Carousel fade:
<div class="carousel carousel-fade" id="carouselExampleCaptions">
Carousel autoplay:
<div class="carousel slide" id="carouselExampleCaptions" data-bs-ride=”true”>
Carousel autoplay after user interaction:
<div class="carousel slide" id="carouselExampleCaptions" data-bs-ride=”carousel”>

Inner carousel (mandatory)
<div class="carousel-inner">
Inner item:
<div class="carousel-item active"> (apply active to only 1 for it to work)

Button to navigate:
<button class="carousel-control-prev" type="button" data-bs-target="#carouselExample" data-bs-slide="prev">
    <span class="carousel-control-prev-icon" aria-hidden="true"></span>
    <span class="visually-hidden">Previous</span>
</button>
<button class="carousel-control-next" type="button" data-bs-target="#carouselExample" data-bs-slide="next">
    <span class="carousel-control-next-icon" aria-hidden="true">>/span>
    <span class="visually-hidden">Next>/span>
</button>

Indicators (show where you at in the slide, should be added before inner carousel)
<div class="carousel-indicators">
    <button type="button" data-bs-target="#carouselExampleIndicators" data-bs-slide-to="0" class="active" aria-current="true" aria-label="Slide 1"></button>
    <button type="button" data-bs-target="#carouselExampleIndicators" data-bs-slide-to="1" aria-label="Slide 2"></button>
    <button type="button" data-bs-target="#carouselExampleIndicators" data-bs-slide-to="2" aria-label="Slide 3"></button>
</div>

Colors

Textfarben: <p class="text-(color)">
Hintergrundfarben: <p class="text-bg-(color)">
color: primary,  success,  info, warning, danger, light, dark, secondary
Wenn man “text” bei “text-bg-(color)“ dazuschreibt, verändert sich die Farbe, je nach Hintergrundfarbe, damit der Text noch sichtbar ist (bei dark, wird der Text weiß)
