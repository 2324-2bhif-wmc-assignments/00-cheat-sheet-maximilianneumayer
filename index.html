Hello World

  1. What is the html head element?
    The <head> element is a container for metadata. (data about the html document, which is not displayed) Elements which can go into the head element:
    • <title> (required in every HTML document)
    • <style>
    • <base>
    • <link>
    • <meta>
    • <script>
    • <noscript>
  2. What does <!DOCTYPE html> mean?

    A html document needs to start with a <!DOCTYPE html> declaration. This isn't a html tag, it is information to the browser which version of HTML the document is.

  3. What are the two top most elements within the html element?

    The most common elements within the html element are:

    <head>

    See here.

    <body>

    The <body> element contains visible content of a webpage. Any element the user can interact with(text, images, links, videos, forms, etc.)

  4. How to emphasize a text and how is it displayed in the browser?

    You can use the <em> element. This element is used to add a semantic meaning to the text and indicate varying degrees of importance or emphasis. The <em> element is used to emphasize text in a way that indicates a slight change in emphasis or importance. By default, it is typically rendered in italics by browsers.

  5. How to mark up important text and how is it displayed in the browser?

    You can use the <mark> element. This element is used to highlight or mark text for emphasis, making it stand out visually. It makes the font-background glow yellow. You can change the color in the CSS document. You can also use the <strong> element. This element is used to emphasise text, so it gets thicker. It makes the text stand out more, by thickening the letters of the text in the <strong> tag.

html info

<!DOCTYPE html>
An html document needs to start with a <!DOCTYPE html > declaration. This isn't a html tag, it is information to the browser about what type of document this is.
<html>
<html> tag represents the root of an HTML document. The <html> tag also contains all HTML-tags except the <!DOCTYPE> tag. In the html tag you can also declare the language with the "lang" attribute which helps search engines and browsers.
<head>
The <head> element is a container for metadata. (data about the html document, which is not displayed) Elements which can go into the head element:
  • <title> (required in every HTML document)
  • <style>
  • <base>
  • <link>
  • <meta>
  • <script>
  • <noscript>
<meta>
The <meta> tag defines metadata about an HTML document. <meta> tags always go in the <head> element. <meta> tags are typically used to specify character set, page description, keywords, author of the document, and viewport settings and are being used by browsers, search engines and other web services. Metadata won't be shown on the website, but it is machine parsable. Attributes from the <meta> tag:
  • charset

    value: character_set
    description: Specifies the character encoding for the HTML document

  • content

    value: text
    description: Specifies the value associated with the http-equiv or name attribute

  • http-equiv

    value: content-security-policy, content-type, default-style, refresh
    description: Provides an HTTP header for the information/value of the content attribute

  • name value: application-name, author, description, generator, keywords, viewport
    description: Specifies a name for the metadata
