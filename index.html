  • Word Wide Web General

    1. What are the most essential technologies that made the World Wide Web such a success?

      URL (uniform resource locator), which is the addressing scheme to find a document. HTTP (hypertext transfer protocol), which connects computers together. HTML (hypertext markup language), which formats pages containing hypertext links. CSS (Cascading Style Sheets) JS (JavaScript) Web Server, Server like facebook.com which can response to a request from a Web Client Web Client, a browser which the user uses to request html-files from the Web Server

    2. How are resources (html pages, images, etc.) addressed in the web?

      URLs. The uniform resource locator (URL) is the most common form of resource identifier. URLs describe the specific location of a resource on a particular server. They tell you exactly how to fetch a resource from a precise, fixed location.

    3. What are the general components of an url?

      An url typically has 9 parts, the protocol (or scheme), subdomain, domain name, top level domain, port, path, query, parameters, and fragment. But the 3 general components of an url, is the scheme, the domain name (server) and the page.

  • html info

    <!DOCTYPE html>
    An html document needs to start with a <!DOCTYPE html > declaration. This isn't a html tag, it is information to the browser about what type of document this is.
    <html>
    <html> tag represents the root of an HTML document. The <html> tag also contains all HTML-tags except the <!DOCTYPE> tag. In the html tag you can also declare the language with the "lang" attribute which helps search engines and browsers.
    <head>
    The <head> element is a container for metadata. (data about the html document, which is not displayed) Typical metadata information: document title, character set, styles, scripts, and other meta information Elements which can go into the head element:
    • <title> (required in every HTML document)
    • <style>
    • <base>
    • <link>
    • <meta>
    • <script>
    • <noscript>
